Tab 3

Tabs can be created using the following markup:

<tab-view>
  <view-box>
    <p>Your content here</p>
  </view-box>
</tab-view>

Tab 4

All tabs must have unique name.

Library sets automatic name via tab "index" but you can supply custom name with "data-name" attribute:

<view-box data-name="my-tab"></view-box>

This tab should be named "my-tab"
